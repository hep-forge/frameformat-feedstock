Feedstock for [frameformat](https://git.ligo.org/kagra/libraries-addons/root/framel) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Frame Format Library

## Install

```bash
conda install -c hep-forge -c conda-forge frameformat
```
